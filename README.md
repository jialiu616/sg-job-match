# SG Job Match 工作配对

Smart job-matching platform for workers and employers in Singapore.
新加坡蓝领人才与雇主智能匹配平台。

## About 关于

This platform connects job seekers (especially manual laborers and service workers) with small-medium business employers through intelligent matching — no complex resume needed.

本平台通过智能匹配，将求职者（特别是体力劳动者和服务业者）与中小企业雇主连接起来——无需复杂简历。

## Features 功能

- 🌐 Bilingual UI (中文/English)
- 📱 Mobile-first responsive design
- 🤖 Smart rule-based matching engine
- ✅ Simple form submission
- 🎯 Match score with detailed breakdown

## Tech Stack 技术栈

- React 19 + TypeScript + Vite
- TailwindCSS v4
- react-i18next (bilingual support)
- Rule-based matching engine

## Getting Started 快速开始

### Prerequisites 前提条件

- Node.js 18+
- npm 9+

### Run Locally 本地运行

```bash
cd sg-job-match
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Build 构建

```bash
npm run build
```

Build output is in the `dist/` folder.
